Add a full-featured Installment Offer Calculator to the underwriting app as part of the Offer Strategy section. This should use real-world underwriting logic and be presented in a professional, friendly way for team use only.

🧠 Core Seller Framework (display as tooltip or helper text):
“We offer sellers 3 things:

Speed

Convenience

Price

But they can only choose 2.”

If they want:

Speed + Convenience → Wholesale Offer

Speed + Price → Subject-To or Seller Finance

Price + Convenience → Installment/Novation Offer (this module)

📥 INPUTS NEEDED:
ARV – After Repair Value (e.g. $200,000)

EstimatedRepairs – e.g. $30,000

NegativeAdjustments – e.g. $0

PositiveAdjustments – e.g. $0

DiscountToSellFast – e.g. $10,000

BuyerOverAskBonus – e.g. $0

ClosingCosts – e.g. $18,400

MinAcceptableProfit – e.g. $15,000

SplitWithSellerBonus – e.g. $0

🧮 CALCULATION STEPS:
js
Copy
Edit
ListPrice = ARV - EstimatedRepairs - NegativeAdjustments + PositiveAdjustments - DiscountToSellFast;

FinalSalesPrice = ListPrice + BuyerOverAskBonus;

GrossAfterClosing = FinalSalesPrice - ClosingCosts;

InstallmentMAO = GrossAfterClosing - MinAcceptableProfit + SplitWithSellerBonus;

Copy
