Add a full Seller Finance Offer Calculator to the Offer Strategy tool. This module should calculate monthly payments, total interest paid, and 5–10 year cash flow using amortization or interest-only logic. It must factor in resale value (ARV), rehab, and capital cost.

📥 REQUIRED INPUTS:
These will be entered by the user or autofilled from underwriting:

🔹 Deal Structure
ARV: e.g. $270,000

SellerFinancePurchasePrice: e.g. $250,000

DownPayment: e.g. $15,000

InterestRate: e.g. 6.5%

AmortizationTermMonths: e.g. 360

BalloonTermMonths: e.g. 60 or 120 (optional)

🔹 Operating Data
MonthlyTaxes: e.g. $150

MonthlyInsurance: e.g. $100

MonthlyHOA: e.g. $0

MonthlyRent: e.g. $2,000

AnnualMaintenanceReserve: e.g. $1,200

ClosingCosts: e.g. $5,000

RehabBudget: e.g. $30,000

CapitalCost: e.g. $10,000

🧠 CORE CALCULATIONS
1. Loan Amount

LoanAmount = SellerFinancePurchasePrice - DownPayment;
2. Monthly Payment (Amortized)
Use amortization formula:

js
Copy
Edit
MonthlyPI = (LoanAmount * r * (1 + r)^n) / ((1 + r)^n - 1);
Where:

r = monthly interest rate (interestRate / 12)

n = total payments (loan term in months)

3. Interest-Only Option (if selected)

MonthlyTotalOutflow = MonthlyPI + MonthlyExpenses;
