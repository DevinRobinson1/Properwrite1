3 ▪ Backend Endpoints
POST /api/jv-deals

Validate payload server-side.

If address already exists in cache (Redis), reuse geocode to cut API cost.

Store deal in jv_deals table with status pending.

Signal (app/events.py): after insert emit jv_deal_submitted event.

Zapier Webhook: send event payload to ZAPIER_JV_HOOK_URL.

Include partner email, phone, address, deal type, pricing.

Admin: extend existing /admin/jv-deals list to add filters (city, state, partner, status). Leverage DataTables server-side pagination.
