🧠 Functionality Overview
1️⃣ Zillow API Integration (RapidAPI)
Use these endpoints:

propertyExtendedSearch with status_type=RecentlySold

Use optional zpid with property endpoint for details

Query & Filter Logic:
Match Criteria:

Sold within last 90 days, expand to 365 days only if < 3 comps

Radius: start at ¼ mile, then ½, then max 1 mile

Same beds, baths, ±500 sqft

Same property type + post-renovation profile

2️⃣ OpenAI API Integration
Use OpenAI API to:

🔧 AI-Powered Comps Enhancer:
Auto-adjust comp prices based on:

Bedroom/bath differences

Proximity to traffic/commercial

Renovation quality

Lot size / pool / carport

Format insight with labels like:

"Adjusted +$15K for additional bedroom"
"Reduced -10% for busy road exposure"

🧠 Natural Language Summary Generator:
Prompt OpenAI to:

Summarize 3–5 best comps

Suggest a target ARV

Output a 1-paragraph plain English breakdown

Highlight concerns or confidence score

Add language like:

“Based on the comps and adjustments, a conservative ARV target would be $327,000. There is moderate confidence due to age of sale (120 days) and proximity.”
