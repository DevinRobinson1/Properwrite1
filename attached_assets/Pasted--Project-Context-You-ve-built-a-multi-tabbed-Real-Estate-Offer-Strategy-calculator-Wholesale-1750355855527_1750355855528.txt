## 🧠 Project Context:
You’ve built a multi-tabbed Real Estate Offer Strategy calculator (Wholesale, Installment, Subject-To, Seller Finance), each with access to the same base property data.

Currently, each tab displays **all strategy types and calculations** — making it repetitive and harder to focus on the specific exit strategy being evaluated.

---

## 🎯 GOAL

Restructure the content logic so that each **tab displays only the components and calculations related to that specific strategy**. The global property data should still be accessible/shared, but only the strategy-relevant blocks should render under each tab.

---

## ✅ IMPLEMENTATION GUIDELINES:

### 🔁 Shared Components Across All Tabs (Global Data)

These components should be shown across all tabs and persist:
- ✅ Property Information / Address / Metrics (ARV, Rent, Sq Ft)
- ✅ Property Details + Data Sources
- ✅ Offer Strategy Summary at bottom (if user wants to compare)

✅ Keep these sections at the top of the screen across all strategies.

---

### 🔀 Tab-Specific View Rules

Modify each tab to show only the related sections:

---

### 📦 Wholesale Tab:
- Show:
  - Wholesale/Cash Offer Calculator
  - Max Profit, ARV, Repair Costs
  - Assignment Price Logic
  - Wholesale Strategy Block in Compare Dashboard

- Hide:
  - Installment Offer sections
  - Seller Finance blocks
  - Subject-To fields (unless user manually toggles them)

---

### 💳 Installment Tab:
- Show:
  - Installment Offer Calculator
  - Seller Psychology Module (Speed, Convenience, Price)
  - Payment Plan + Monthly Breakdown
  - Installment Strategy block in Compare Dashboard

- Hide:
  - Wholesale offer boxes
  - Subject-To and Seller Finance components

---

### 🔄 Subject-To Tab:
- Show:
  - Subject-To Deal Structure
  - Takeover Terms, Existing Loan, Wrap Equity
  - Entry Fee, Out-of-Pocket, MAO
  - Subject-To Strategy Summary in Compare block

- Hide:
  - Cash offer logic
  - Installment or Seller Financing tools

---

### 🧾 Seller Finance Tab:
- Show:
  - Seller Finance Calculator
  - Interest Rate + Balloon Payment Logic
  - Seller Finance Offer Summary
  - Seller Strategy row in Compare Dashboard

- Hide:
  - Subject-To assumptions
  - Wholesale breakdowns

---

## 🔁 Compare Offers Dashboard (Optional View)

- ✅ If the user wants to **compare across all strategies**, show the Compare Offers Dashboard at the **bottom of all strategy pages**
- ✅ If they only want to evaluate one strategy, allow them to toggle "Compare View: On/Off"

---

## ⚠️ DO NOT

- ❌ Do not remove calculations or delete any logic
- ❌ Do not merge all tabs into a single strategy view
- ✅ Only filter and show the data relevant to each strategy per tab

---

## ✅ FINAL OUTPUT

- Each tab (Wholesale, Installment, Subject-To, Seller Finance) shows only strategy-relevant blocks
- Global property data stays visible at the top
- Compare Dashboard is optional/togglable
- UX is simplified and more actionable for investors evaluating one deal at a time
