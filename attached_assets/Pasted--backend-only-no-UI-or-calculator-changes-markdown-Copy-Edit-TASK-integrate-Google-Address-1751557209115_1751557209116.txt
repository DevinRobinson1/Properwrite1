🔧 **TASK: integrate Google Address Validation API to hard-validate every
address before underwriting.  Do NOT modify Acquisitions/Dispositions UI.**

### 1. Endpoint
POST https://addressvalidation.googleapis.com/v1:validateAddress
Headers:
  Content-Type: application/json
  X-Goog-Api-Key: $GMAPS_API_KEY

### 2. Request payload
{
  "address": { "regionCode":"US", "languageCode":"en", "postalCode": zip,
               "administrativeArea": state, "locality": city,
               "addressLines": [street] },
  "enableUspsCass": true           // for highest accuracy in US
}

### 3. Success rule
Accept only responses where:
response.verdict.hasUnconfirmedComponents == false
&& response.verdict.hasInferredComponents == false
&& response.verdict.addressComplete == true
