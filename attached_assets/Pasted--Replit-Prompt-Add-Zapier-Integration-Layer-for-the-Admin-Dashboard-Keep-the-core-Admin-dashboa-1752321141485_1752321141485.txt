Implementation

// /services/zapierWebhook.ts
export async function fireZapier(triggerName: string, payload: any) {
  const url = process.env[`ZAPIER_HOOK_${triggerName}`]; // store each Zap URL in .env
  if (!url) return;
  await fetch(url, { method: "POST", headers: {'Content-Type':'application/json'}, body: JSON.stringify(payload) });
}

Call fireZapier('NEW_USER_SIGNED_UP', payload) inside existing user-creation handler.

3. Inbound Actions (from Zapier)

Zapier Action	HTTP Endpoint (admin auth)	Purpose
Grant Bonus Credits	POST /admin/zap/credits/grant {user_id,credits}	Adds credits, logs to credit_ledger
Approve JV Deal	POST /admin/zap/jv/approve {deal_id}	Changes status → Approved, fires outbound “JV Approved” trigger
Send Message to User	POST /admin/zap/notify {user_id,message}	Inserts into notifications table (in-app toast + email)

All endpoints require a header:

X-ZAP-SECRET: ${process.env.ZAPIER_SHARED_SECRET}
