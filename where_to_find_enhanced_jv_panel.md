# How to Access Enhanced JV Admin Panel

## Steps:
1. In the admin dashboard, click on the **"JV Deals"** tab (not "Affiliates")
2. You'll see a blue highlighted section at the top that says "Enhanced JV Admin Panel"
3. Click the **"Open Enhanced Panel"** button

## What you'll get:
- Advanced sortable data tables
- User portfolio management drawer
- Comprehensive deal detail modals
- Enhanced search and filter capabilities
- Real-time metrics and charts

## Current URL Structure:
- Basic JV view: `/admin` (JV Deals tab)
- Enhanced JV panel: `/admin/jv-deals-enhanced`

You can also go directly to: `/admin/jv-deals-enhanced`
